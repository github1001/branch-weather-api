# 🌦️ Branch Weather API

A full-stack portfolio project demonstrating an enterprise-style weather dashboard built with **Spring Boot**, **SQL Server**, **Tailwind CSS**, and the **Open-Meteo API**.

The repository consists of:

```
branch-weather-api/

├── branch-api/      Spring Boot Backend
├── ui/              Tailwind CSS Frontend
├── demo/            Screenshots & GIFs
└── Weather API.postman_collection.json
```

---

# Project Overview

This project demonstrates:

- RESTful API Development

# Running the Backend

```bash
cd branch-api

mvn spring-boot:run
```

Backend URL

```
http://localhost:8080
```

More details:

➡️ [branch-api/README.md](branch-api/README.md)

---

# Running the Frontend

Open another terminal

```bash
cd ui

npx serve .
```

Frontend URL

```
http://localhost:3000
```

More details:

➡️ [ui/README.md](ui/README.md)
